🏠 House Price Prediction Web App
A complete end-to-end Machine Learning web application that predicts house prices based on user input such as area, number of bedrooms, and bathrooms.
This project demonstrates the full lifecycle of a Machine Learning product — from data preprocessing and model training to deployment using a web interface.
🚀 Features
Predict house prices instantly
Trained using real housing dataset
Fully responsive professional UI
Flask-based backend integration
Machine Learning model using Scikit-learn
Clean and structured project architecture
Ready for deployment
