House Price Prediction Web App
Overview

The House Price Prediction Web App is an end-to-end machine learning application that predicts house prices based on user inputs such as area, number of bedrooms, and bathrooms.

This project demonstrates the full lifecycle of a machine learning system, including data preprocessing, model training, model saving, and deployment using a web application.

The system allows users to enter house details through a web interface and receive an instant price prediction generated by a trained machine learning model.

Features

Machine Learning Prediction
Predicts house prices using a trained regression model.

Interactive Web Interface
Simple and responsive form for entering property details.

Instant Prediction
Users receive predicted house prices immediately after submitting the form.

Clean Project Architecture
Well organized folders for data, models, frontend, and backend code.

Flask Web Integration
Backend application built using the Flask web framework.

Deployment Ready
Designed in a way that can easily be deployed to cloud platforms.

Dataset

This project uses a housing dataset for training the machine learning model.

The dataset contains information such as:

Area (square feet)

Number of Bedrooms

Number of Bathrooms

House Price

These features are used to train the model to predict house prices.

Installation
1 Clone the Repository
git clone https://github.com/yourusername/house-price-prediction.git
cd house-price-prediction
2 Install Dependencies
pip install -r requirements.txt
3 Train the Machine Learning Model
python train_model.py

This will create the trained model file:

model/house_price_model.pkl
4 Run the Web Application
python app.py
5 Open in Browser
http://127.0.0.1:5000
